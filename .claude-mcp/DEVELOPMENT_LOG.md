# FOR-NEWS 開發日誌

## 專案概述
半自動化BBC新聞聚合與內容擷取系統

## 技術架構
- **前端**: Next.js 15.5.4 + React 19.1.0
- **RSS解析**: rss-parser
- **內容擷取**: @postlight/mercury-parser
- **新聞源**: BBC (5個頻道)
- **記憶增強**: MCP (Model Context Protocol)
